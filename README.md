## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in the top right)
3. Click "Load unpacked"
4. Select this directory
5. Open a new tab to see the custom page

## Features

- Time Display with editable positions
- Toggleable search bar that redirects to your search engine of choice
- Customizable background WITH adjustable blur effect.
- Clean, minimal design
